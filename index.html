Foundations of Interactive Design

What is this class about?

In Foundations of Interactive Design, you will learn how to create your own websites and learn how to create your very own games!

HTML

HTML will be the first thing you learn in this class. HTML stands for Hyper Text Markup Language. HTML is a markup language that is used to build over 90% of the pages you visit everyday! This website you are on now is built using HTML, CSS, and Javascript!

CSS

After gaining a solid foundation of HTML, we will start learning CSS. CSS stands for Cascading Style Sheets. CSS is how you style your webpages and make them look great! Click on the button below to see what CSS can do for your website.



Javascript

JavaScript is a scripting or programming language that allows you to implement complex features on web pages — every time a web page does more than just sit there and display static information for you to look at — displaying timely ontent updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc. — you can bet that JavaScript is probably involved.
